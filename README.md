# Agent-Based Disease Spread Simulation
My modeling course term project


CURRENTLY IMPLEMENTED
- Agent-based simulation of disease spread using the SIR model  
- Classes for `Agent`, `DiseaseModel`, and `Simulation`  
- Random movement of agents in a 2D environment  
- Infection spread based on proximity and probability  
- Recovery after a defined number of steps  
- Data collection for Susceptible, Infected, Recovered counts  
- Visualization of SIR curves using Matplotlib  


WORK IN PROGRESS
- Implement additional mitigation strategies (e.g., vaccination)  
- Add more configurable parameters (agent behavior, movement patterns)  
- Enhance visualization with real-time animation  
- Optional end condition based on zero infected agents
